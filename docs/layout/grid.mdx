# Grid

The Grid component implements a layout that arranges child components in a grid.

## Example

```jsx live
<Grid rows={2} columns={3} rowGap='10px' columnGap='10px'>
  <BorderBox>Item1</BorderBox>
  <BorderBox>Item2</BorderBox>
  <BorderBox>Item3</BorderBox>
  <BorderBox>Item4</BorderBox>
  <BorderBox>Item5</BorderBox>
  <BorderBox>Item6</BorderBox>
</Grid>
```

### Grid Template

The default value of row/columnTrackList is `1fr`.

- grid-template-rows: `repeat(rows, rowTrackList)`

- grid-template-columns: `repeat(columns, columnTrackList)`

```jsx live
<Grid
  rows={2}
  columns={2}
  columnTrackList='150px 200px'
  rowTrackList='80px'
  rowGap='10px'
  columnGap='10px'
>
  <BorderBox>Item1</BorderBox>
  <BorderBox>Item2</BorderBox>
  <BorderBox>Item3</BorderBox>
  <BorderBox>Item4</BorderBox>
  <BorderBox>Item5</BorderBox>
  <BorderBox>Item6</BorderBox>
  <BorderBox>Item7</BorderBox>
  <BorderBox>Item8</BorderBox>
</Grid>
```

### Gap

You can set the gap for rows and columns respectively.

```jsx live
<Grid rows={2} columns={3} rowGap='20px' columnGap='10px'>
  <BorderBox>Item1</BorderBox>
  <BorderBox>Item2</BorderBox>
  <BorderBox>Item3</BorderBox>
  <BorderBox>Item4</BorderBox>
  <BorderBox>Item5</BorderBox>
  <BorderBox>Item6</BorderBox>
</Grid>
```

#### Responsive Column

You can set the responsive Column of Grid.

\*only **Column** possible

```jsx live
<Grid
  columns={{ xs: 1, sm: 2, md: 3, lg: 4, xl: 6 }}
  columnGap='20px'
  rowGap='20px'
>
  <BorderBox>Item1</BorderBox>
  <BorderBox>Item2</BorderBox>
  <BorderBox>Item3</BorderBox>
  <BorderBox>Item4</BorderBox>
  <BorderBox>Item5</BorderBox>
  <BorderBox>Item6</BorderBox>
</Grid>
```

Or you can set partial of responsive breakpoints.

```jsx live
<Grid columns={{ xs: 1, md: 3, xl: 5 }} columnGap='20px' rowGap='20px'>
  <BorderBox>Item1</BorderBox>
  <BorderBox>Item2</BorderBox>
  <BorderBox>Item3</BorderBox>
  <BorderBox>Item4</BorderBox>
  <BorderBox>Item5</BorderBox>
  <BorderBox>Item6</BorderBox>
</Grid>
```
