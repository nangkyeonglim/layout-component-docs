# Container

The container **centers your content horizontally**. It's the most basic layout element.

## Example

```jsx live
<Container maxWidth='500px' bgColor='#cfe8fc' padding='1rem'>
  Container
</Container>
```

### Color

You can specify background color and text color for Container.

```jsx live
<Container maxWidth='500px' bgColor='#32d8b4' color='#FFF' padding='1rem'>
  Container
</Container>
```

### Border

A border can be specified for Container.

```jsx live
<Container
  maxWidth='500px'
  border='2px solid #32d8b4'
  borderRadius='8px'
  padding='1rem'
>
  Container
</Container>
```

### Padding & Margin

You can set padding and margin on Container respectively.

```jsx live
<Container
  maxWidth='500px'
  bgColor='#cfe8fc'
  padding='1rem 3rem'
  margin='40px auto'
>
  Container
</Container>
```
